Table Demo

A table is a way of creating a structured grid display of data. The tags we will be interested in are:

  • <table> : identifies the table
  • <thead> : identifies the header row of the table
  • <th> : identifies the text in the head of the collum
  • <tbody> : identifies the body of the table (detail rows)
  • <tr> : identifies the start of the row
  • <td> : identifies the data cell on the row
  • <tfoot> : identifies the footer row of the table

All tags are paired tags (Opening and closing tag). The border parameter on the table tag will allow you to specify a border around your table. The colspan tag and rowspan tag used on the <td> tag allows you to merge collums and rows on a single row and collum cells across rows.
